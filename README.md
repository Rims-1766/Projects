Contact Management System (C)

A Contact Management System built in C language using a Linked List data structure.
This program allows users to store, manage, and organize contacts efficiently through a menu-driven console interface.

The system supports adding, searching, deleting, and categorizing contacts into groups such as Friends and Family, along with marking important contacts as Favourites.

Features

Add new contacts with Name, Phone, Email, and Group

Display all saved contacts

Categorize contacts into Friends and Family

Search contacts by name

Delete contacts

Mark or unmark contacts as Favourite

Display all favourite contacts

Colored terminal output for better readability

Menu-driven interface for easy interaction

Technologies Used

Programming Language: C

Concepts Used:

Linked Lists

Dynamic Memory Allocation (malloc)

Structures

String Handling

Console-based User Interface

Classy Cart – Fashion Budget Checker

Classy Cart is a Python GUI application built using Tkinter that helps users find fashion items within their budget.
The program allows users to select a fashion category such as Clothes, Shoes, Accessories, or Skincare, enter their budget, and view the items they can afford along with their images.

This project demonstrates GUI development, image handling, and budget-based filtering using Python.

Features

User-friendly Graphical User Interface (GUI) using Tkinter

Select fashion categories:

Clothes

Shoes

Accessories

Skincare

Enter a budget to check affordable items

Displays item names and prices

Opens images of affordable products in popup windows

Uses soft pastel UI colors for an attractive interface

Error handling for invalid budget inputs

Technologies Used

Programming Language: Python

Libraries Used:

Tkinter (GUI development)

PIL / Pillow (Image handling)

Concepts Used:

Event-driven programming

GUI widgets (Buttons, Labels, RadioButtons, Entry fields)

Dictionary data structures

Image processing
